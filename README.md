# 📖 [BLOGAPP] <a name="about-project"></a>

**[Blogapp]** is blog application written using Rails and Postgresql

## 🛠 Built With <a name="built-with"></a> 

- Ruby v3.2.2
- Rails v7.0.5
- Postgresql v15.3

In order to run this project you need:
